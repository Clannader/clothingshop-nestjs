# 发布包版本步骤

---

## 1. 发布流程

1. 使用 dev 分支打 tag，上传到 github，通过 github 生成 release note 和 release 分支
2. 进行打包生成最新代码
3. 删除 `package.json` 的 `private`、`devDependencies`、`jest` 节点
4. 运行 `publish:npm`，如果有报错使用 cmd 运行以下命令重新登录验证，再次运行 `publish:npm`

```bash
npm login
```

5. 撤回刚才修改的 `package.json`，修改 `package.json` 的版本号到下一个版本
6. cmd 运行以下命令更新 `package-lock.json`，然后提交代码即可

```bash
npm uninstall all
```
